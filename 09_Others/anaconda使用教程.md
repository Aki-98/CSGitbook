进入annaconda

```
 source ~/anaconda3/bin/activate r
```

查看当前所有环境

```python3
conda info --envs
```

创建环境

```python3
conda create -n 环境名
```

激活环境

```text
Conda activate 环境名
Source activate 环境名
activate + 环境名
```

删除环境

```text
conda remove -n 环境名 --all
```
