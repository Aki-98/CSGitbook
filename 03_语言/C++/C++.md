## 指针

指针的大小：32位 4字节  64位8字节

 

空指针：

空指针指向的内存不可访问，指向编号为0的空间

0-255之间的内存编号是不可以访问的

 

野指针：指向非法内存空间

 

CONST 修饰指针（看const后面跟的是啥）

1.const修饰指针：常量（的）指针

const int * p = &a  （int* p指向的值是个常量）

指针的指向可以改，但指针指向的值不能改

2.const修饰常量：指针（是）常量

指针的指向不可以改，但指针指向的值可以改。

int * const p = &a (p的指向是个常量)

3.const既修饰指针，又修饰常量

const int * const p = &a
